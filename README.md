
# :thought_balloon: markdown cheat sheet 
| task | markdown |
|:---:| :---:|
| heading | # , ## or ###|
| bold | ** bold text ** |
| italic | * italicised text * |
| code | "`code` "|
| horizontal rule | --- |
| link | [title] (https link) |
| image | ![alt text] (image.jpg) |
| footnote | [^1] |
| highight | I need to highlight these ==very important words==. |
| strikethroughr | ~~ the world is flat ~~|
| emoji | :joy: : joy :|
